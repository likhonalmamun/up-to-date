Question 1.

What are the differences between var , let and const in JS ?!

Var is a variable of functional scope But let and const are block scoped .
Var can be updated or redeclared , let can be updated but cannot be redeclared and const cannot be updated or redecleared .
Var has a default value 'undefined'. So it can be accesed without initialization but let or const will give refference error .

Question 2.

What are the differences between arrow function and regular function in JS ?!

Arrow function is declared like a variable and regular function is declared with the function keyword .
Generally regular function can have duplicate parameters but arrow function can not .
In arrow function , value of this keyword is relative to outer function but in regular function , the value is dynamic .

Question 3.

What are the differences among map ,forEach,filter and find methods ?!

Map : It runs a function on each element of an array and returns the results in a new array .
forEach : It runs a function on each element of an array but it returns nothing.
Filter : It checks the given condition on each element of an array and returns the mathing results in a new array .
Find : It checks the given condition on each element of an array and returns the first mathing result .

Question 4.

Why should you use template string ?!

Template strings are a powerful feature of ES6 . It lets us insert/interpolate variables and expressions into strings without concatenating like in older versions of JavaScript. It allows us to create strings that can contain dynamic elements. We can write multiline string without any method
